# Jeongjin.dev – 인터랙티브 포트폴리오

AISW 전공생 **이정진**의 인터랙티브 포트폴리오 랜딩 페이지입니다.  
웹 개발 공부를 하면서 만든 개인 소개 페이지로, 간단한 자기소개와 보유 스킬, 그리고 실제로 제작한 웹페이지들을 한 곳에서 볼 수 있도록 구성했습니다.

---

## 🧷 주요 기능

- **반응형 레이아웃**
  - PC, 태블릿, 모바일 화면 크기에 따라 자연스럽게 배치가 바뀌도록 구현

- **캔버스 배경 애니메이션**
  - `canvas`를 이용해 움직이는 점과 선이 이어지는 네트워크 형태의 배경 효과 구현
  - 핑크 톤의 컬러를 사용해 전체 분위기와 통일감 있게 디자인

- **타이핑 효과 (Typewriter Effect)**
  - “AI Software 전공생 / Web Developer 꿈나무 / 프론트엔드 개발 공부 중” 같은 문구가 한 글자씩 타이핑되는 애니메이션

- **스크롤 등장 애니메이션 (Scroll Reveal)**
  - About / Skills / Projects 섹션이 스크롤에 따라 부드럽게 나타나는 효과

- **프로젝트 카드 & 외부 링크 연동**
  - 각 프로젝트 카드를 클릭하면 새 탭에서 실제 웹페이지로 이동
    - 오늘의 운세 & 타로 카드 웹페이지  
      👉 `https://lsch4435-code.github.io/fortune-tarot/`
    - 개인 포트폴리오 웹사이트  
      👉 `https://lsch4435-code.github.io/lsch4435.1234/`
    - 크리스마스 트리 꾸미기 웹페이지  
      👉 `https://christmastree-dccfb.web.app/index.html`

- **프로필 이미지**
  - 히어로 섹션 오른쪽에 말랑말랑한 모양으로 변화하는 프로필 영역
  - `image/profile.png` 파일을 교체하면 본인 사진으로 쉽게 변경 가능

---

## 🛠 사용 기술

- **HTML5** – 페이지 구조 및 섹션 구성
- **CSS3** – 레이아웃, 반응형 디자인, 글래스모피즘 카드, 애니메이션 스타일
- **JavaScript (Vanilla JS)**  
  - 캔버스 배경 애니메이션
  - 타이핑 효과
  - 스크롤 리빌 애니메이션
  - 모바일 내비게이션 토글
- **Font Awesome** – 아이콘 사용 (메뉴, 스킬 아이콘, 소셜 아이콘 등)
